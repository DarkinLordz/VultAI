# VultAI
VultAI is a Python-based AI chatbot with CustomTkinter GUI.
It's manually set-up, you must enter your API key, Endpoint, AI model.
## Requirements
- Python 3.13.5 +
- External libraries:
- customtkinter
- requests
- pillow
### Branches
The "main" branch is reserved for trusted friends.
Every contributor may contribute to the "contributors".
